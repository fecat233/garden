# 安装
## 预安装
要想使用这个主题，首先需要安装[Zsh](https://www.zsh.org/)和[ohmyzsh](https://github.com/ohmyzsh/ohmyzsh)
## 安装步骤
1. 克隆仓库
```bash
git clone https://github.com/fecat233/garden.git
```
2. 配置.zshrc文件
```json
ZSH_CUSTOM=$HOME/xxx/garden
ZSH_THEME="garden"
```
3. 使.zshrc生效
```bash
source ~/.zshrc
```
